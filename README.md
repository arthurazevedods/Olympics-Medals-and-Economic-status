# Olympic Countries Dataset Dashboard

Este é um aplicativo Dash que visualiza um conjunto de dados contendo informações sobre países participantes das Olimpíadas. O aplicativo permite que os usuários explorem os dados, visualizem o número de medalhas ganhas e obtenham detalhes sobre cada país.

## Recursos

- Exibe um gráfico de barras com o número de medalhas de ouro, prata e bronze ganhas por cada país.
- Permite selecionar um país através de um menu suspenso.
- Apresenta informações detalhadas sobre o país selecionado, incluindo:
  - Código do país
  - Número de medalhas de ouro, prata e bronze
  - Total de medalhas
  - PIB
  - Ano do PIB
  - População

## Tecnologias Utilizadas

- [Python](https://www.python.org/)
- [Dash](https://dash.plotly.com/)
- [Pandas](https://pandas.pydata.org/)
- [Plotly Express](https://plotly.com/python/plotly-express/)
